JavaScript Conditionals

JavaScript Conditional statements allow to perform different actions for different conditions or events. Conditional statements are used to make decisions in JavaScript depending on the outcome of a condition on true false values.

Types of conditional Statements:

  1. if
  2. if else
  3. if else if else
  4. switch
  5. ternary operator

when to use each conditional Statements:

  1. Use if to specify a block of code to be executed if a condition is true
  2. Use if else to specify a block of code to be executed if a condition is true, and another block if it is false
  3. Use if else if else to specify multiple conditions and execute different blocks of code for each condition
  4. Use switch to specify multiple conditions and execute different blocks of code for each condition
  5. Use ternary operator to assign a value based on a condition in a single line

Common Data Types Explained

Strings

Strings are used to represent text in JavaScript. They can be created using single quotes (' '), double quotes (" "), or backticks (` `) for template literals.

Numbers

Numbers in JavaScript can be integers or floating-point values. JavaScript uses the IEEE 754 standard for representing numbers.
