Intro to JavaScript Programming

JavaScript is a loosely typed programming language that can be ran in multiple ways: Via a .js file or directly in HTML using the <script> tag; otherways to run Javascript is with NodeJS, The same one used in the embded Content

Syntax

Syntax are the rules of the programming Language, and explains how programs must be made

Javascript values and literals

Values are the data that a program uses, while literals are the actual representations of those values in code. For example, a number literal is 42, and a string literal is "Hello".
