JavaScript Operators

Operators are symbols that perform operations on values and variables. They are used to perform arithmetic, comparison, logical, and assignment operations.

JavaScript Arithmetic/Math Operators

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Modulus (Remainder): %
  • Exponentiation: **
  • Increment: ++
  • Decrement: --

+ can also be used for concatenation of strings, for more on that look at the String section

JavaScript assignment Operators

Assignment operators are used to assign values to variables. The most common assignment operator is the equal sign (=).

  • Assignment: =
  • Addition assignment: +=
  • Subtraction assignment: -=
  • Multiplication assignment: *=
  • Division assignment: /=
  • Modulus assignment: %=
  • Modulus assignment: %=

Logical Assignment Operators

Logical assignment operators combine logical operations with assignment. They allow you to assign a value to a variable based on a logical condition.

  • Logical AND assignment: &&=
  • Logical OR assignment: ||=
  • Logical nullish assignment: ??=

Comparison Operators

  • Equal: ==
  • Not equal: !=
  • Strict equal: ===
  • Strict not equal: !==
  • Greater than: >
  • Less than: <
  • Greater than or equal to: >=
  • Less than or equal to: <=
